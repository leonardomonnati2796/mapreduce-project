System Control Panel
Dynamic Cluster Management: Use these buttons to dynamically manage the MapReduce cluster.
  • Add Master: Adds a new master to the cluster and triggers a new leader election
  • Add Worker: Adds a new worker to increase processing capacity
  • Reset Cluster: Restarts the cluster with default configuration (3 masters, 3 workers)
  • Stop All: Stops all cluster services
All actions are executed through the docker-manager.ps1 script and are logged in real-time.
